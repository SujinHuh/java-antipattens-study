# Antigravity Rules

## 1. 문서 작성 포맷팅 규칙 (가로 스크롤 방지)
- **강제 줄바꿈(Hard Wrap) 필수 적용**:
  사용자가 CLI 터미널 환경이나 화면이 좁은 편집기 환경에서도 가로 스크롤을 하지 않고 편하게 한눈에 텍스트를 읽을 수 있도록 합니다.
- **적용 대상**: `review-answer.md` 및 모든 `docs/` 하위 마크다운 문서 내의 긴 한글 텍스트 문단.
- **적용 방식**: 한 문장 또는 문단이 **한 줄당 약 60~80자 내외**가 되면 어미나 쉼표(,)를 기준으로 명시적으로 줄바꿈(Hard Wrap)을 넣어 작성합니다.
