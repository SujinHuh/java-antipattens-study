# Antipattern Kit

Java/Spring Boot 면접 대비 과정에서 반복해서 발견한 안티패턴을 모아, 나중에 AI에게 줄 수 있는 코딩 지침으로 발전시키는 공간이다.

목표는 단순 복습이 아니라, AI가 코드를 생성할 때 피해야 할 패턴과 지켜야 할 기준을 명확히 알려주는 것이다.

## 사용 방식

1. 문제 풀이 중 반복해서 나온 안티패턴을 기록한다.
2. 나쁜 코드 예시와 위험성을 정리한다.
3. 면접에서 설명할 문장을 만든다.
4. 좋은 코드 방향을 정리한다.
5. AI 에이전트에게 줄 코딩 지침으로 바꾼다.
