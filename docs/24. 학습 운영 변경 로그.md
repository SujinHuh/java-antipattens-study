# 24. 학습 운영 변경 로그

이 문서는 면접 대비 학습을 진행하면서 docs 구조, 문제 출제 기준, 피드백 방식이 왜 바뀌었는지 기록한다.

문제 풀이 결과와 TIL은 `6. TIL-로그 에이전트.md`, 회차 상태는 `17. 루프 상태 기록.md`, 개념별 출제 상태는 `23. 개념별 출제 확장 매트릭스.md`에 남긴다.
이 문서는 그중에서도 "문서와 운영 방식 자체가 어떻게 개선되었는지"를 기록하는 용도다.

## 기록 원칙

- 사용자의 학습 방식이 바뀌어 docs를 수정한 경우 기록한다.
- 문제 출제 기준, 힌트 제공 방식, 채점 방식, 검수 흐름이 바뀐 경우 기록한다.
- 문서를 분리하거나 이름을 바꾼 경우 기존 위치와 새 위치를 함께 적는다.
- 단순 오타 수정은 필요할 때만 기록한다.
- 기록은 길게 쓰지 말고, 나중에 봤을 때 왜 바뀌었는지 알 수 있을 정도로만 남긴다.

## 기록 형식

```text
날짜:

변경 이유:
변경 내용:
영향:
다음에 확인할 것:
```
