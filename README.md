# Java Antipatterns Interview Lab

Java/Spring Boot 백엔드 면접 대비를 위한 코드 리뷰형 학습 공간이다.

## 구조

- `docs/`: 면접 대비 학습 루프, 에이전트 프롬프트, 답변 템플릿, 상태 기록
- `problems/`: 회차별 문제 코드, 답변, 피드백, TIL
- `antipattern-kit/`: 반복해서 나온 Java/Spring 안티패턴을 AI 코딩 지침으로 정리하는 공간

## 시작 순서

1. `docs/0. 면접 대비 루프 엔지니어링 시스템 프롬프트.md`로 전체 원칙을 확인한다.
2. `docs/13. 실제 루프 실행 프롬프트.md`로 문제 풀이를 시작한다.
3. 문제별 코드는 `problems/` 아래에 회차별 폴더로 생성한다.
4. 반복해서 나온 안티패턴은 `antipattern-kit/`에 코딩 지침으로 정리한다.
